- Argumentos padrões: Argumentos passados, de forma posicional, para não ser obrigatório sua passagem, mas ele já é definido.
void (int i = 0, int b=0, int c){}

- objetos const: Objetos que não podem ser modificados, mas podem ser lidos.
  - Garante que um método não modifique uma variável const, em compilação já é feito.
